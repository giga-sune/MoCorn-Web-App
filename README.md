# MoCorn-Web-App

**A Secure Media Management System**

MoCorn is a full-stack web application designed for managing a library of movies and TV shows. Built with a focus on backend security and data integrity, it features administrative controls and a centralized NoSQL database.

 Tech Stack
- **Frontend:** EJS (Embedded JavaScript), CSS (UI in active iteration)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose ODM)
- **Security:** Session-based Authentication, Server-side Validation, Bcrypt password hashing

Key Features
- **Role-Based Access:** Protected routes for Admin users to Create, Update, and Delete media.
- **Secure Authentication:** User registration and login with encrypted credentials.
- **Dynamic Content:** Real-time data fetching from MongoDB.
- **Server-Side Validation:** Custom validation logic to ensure data quality before entry.
 Project Status

The core backend architecture, security protocols, and administrative dashboards are fully functional. **The UI is currently in an active development phase (v1.5).**
